MIKROTIK BONDING INTERFACE / LINK AGGREGATION MODE BALANCE-RR

Bonding is a technology that allows the aggregation of more than one ethernet interface and combine them into one virtual link so that we will get greater bandwidth throughput. as an example your router ether1 only support 100Mbps, you can bonding with ether2 and finaly you can get more speed to 200Mbps.

MIKROTIK - A





Copy-Paste in Terminal Mikrotik - A
######################################
# BONDING INTERFACE MIKROTIK-A
# Created By: BuanaNETPBun.Github.io
######################################
 /interface bonding
add name="bonding" link-monitoring=arp arp-ip-targets="192.168.50.2" mode=balance-rr slaves="ether1,ether2" comment="By BNT"
/ip address
add address="192.168.50.2/24" interface="bonding" comment="By BNT"

# BONDING INTERFACE MIKROTIK-B
